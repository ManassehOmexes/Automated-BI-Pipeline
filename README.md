# Automated BI-Pipeline System

Ein automatisiertes Business Intelligence Pipeline-System für E-Commerce Daten mit Machine Learning Forecasting.

## Projektbeschreibung

Dieses Projekt entwickelt eine vollautomatische BI-Pipeline, die:

- Daten aus verschiedenen Quellen importiert und bereinigt
- Automatisierte Forecasts mit Machine Learning erstellt
- Interaktive Dashboards für Business Intelligence bereitstellt
- OpenAI Integration für natürlichsprachliche Insights nutzt

## Tech Stack

- **Programmiersprache:** Python 3.11+
- **Datenbank:** PostgreSQL
- **Containerisierung:** Docker
- **Workflow-Automation:** n8n
- **ML Framework:** Prophet, MLflow
- **Visualisierung:** Metabase
- **Cloud:** AWS
- **CI/CD:** GitHub Actions
